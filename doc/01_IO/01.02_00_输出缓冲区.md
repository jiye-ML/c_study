  * printf标准函数把要显示的内容放在输出缓冲区里,计算机在合适的时候把输出缓冲区里的内容显示在屏幕上在以下四种情况下计算机会把输出缓冲区里的内容显示在屏幕上
    1. 如果输出缓冲区里包含'\n'字符就把
        它前面的内容显示在屏幕上
    2. 程序结束后计算机会把程序放在输出
       缓冲区里的内容显示在屏幕上
    3. 当输出缓冲区被充满的时候它里面的内容
       会显示在屏幕上
    4. 可以使用fflush(stdout)语句强制把输出
       缓冲区里的内容显示在屏幕上

```
/*
 * 输出缓冲区演示
 * */
#include <stdio.h>
int main() {
    printf("1");
    fflush(stdout);
    while (1) {
    }
    return 0;
}
```
